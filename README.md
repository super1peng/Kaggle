# Kaggle

kaggle -- 房价预测

使用 vscode 进行可以石化操作

## 第一次（初始化）
clone 自己的repo
git clone https://github.com/super1peng/Kaggle.git

## 进入该仓库文件夹
cd kaggle

## 查看该仓库远程repo
git remote

## 添加 远程 repo 仓库（添加一次之后就不用再进行添加了）
git remote add origin_online https://github.com/super1peng/Kaggle.git

## 第二次（修改并进行提交）
git pull origin_online matser

## 上传到自己的repo仓库
git push origin master
